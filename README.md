# :japanese_castle:Japanese Holiday API
## Overview
Web api for listing all Japanese Holidays and Checking if a certain date is a holiday.
Holidays are based on http://www8.cao.go.jp/chosei/shukujitsu/syukujitsu.csv.
## Request
```
{
    type: "list | check"
    yy: <default-current>
    mm: <default-current>
    dd: <default-today>
}
```
